## Reporting a Vulnerability

If you discover a security vulnerability in GrantFlow, please report it responsibly by following these steps:

1. **Do NOT open a public issue** about the vulnerability.
2. Send a detailed email to [INSERT SECURITY CONTACT EMAIL HERE] with:
   - A clear description of the vulnerability
   - Steps to reproduce the issue
   - Potential impact
   - Any suggested fixes
3. Wait for a response from the maintainers before disclosing any details publicly.

We will respond to your report as quickly as possible and keep you updated on the progress of a fix.

Thank you for helping keep GrantFlow secure!
